## Integration Setup

### Quick Setup Process

<Steps>
  <Step title="Connect Platform">
    **Account Linking**
    - Navigate to Integrations settings
    - Select project management tool
    - Authorize Optiverse access
    - Choose workspace/organization
  </Step>
  
  <Step title="Configure Mapping">
    **Data Synchronization**
    - Map meeting types to project categories
    - Set up action item → task conversion
    - Define priority mappings
    - Configure team assignments
  </Step>
  
  <Step title="Enable Automation">
    **Workflow Rules**
    - Auto-create tasks from action items
    - Sync meeting outcomes to projects
    - Set up notification preferences
    - Test integration functionality
  </Step>
</Steps>

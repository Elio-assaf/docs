## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="AI Not Joining Meetings">
    **Possible Causes:**
    - Calendar permissions insufficient
    - Meet integration disabled
    - Meeting privacy settings
    - Network connectivity issues
    
    **Solutions:**
    - Reconnect Google account
    - Check integration settings
    - Verify meeting permissions
    - Test network connection
  </Accordion>
  
  <Accordion title="Poor Audio Quality">
    **Audio Issues:**
    - Background noise interference
    - Multiple speakers talking
    - Poor internet connection
    - Microphone quality issues
    
    **Solutions:**
    - Use noise cancellation
    - Encourage turn-taking
    - Check bandwidth
    - Recommend better microphones
  </Accordion>
  
  <Accordion title="Recording Problems">
    **Recording Issues:**
